# CacheCow - Download Videos Your Way

## Description
Cross-platform application built with Tauri that downloads videos from various social media sites on a schedule. Built on yt-dlp, avoid the algorithm and watch where, when and how you want. Available for macOS, Windows, and Linux with native installers and an auto-updating release pipeline via GitHub Actions.

## Details
- Built cross-platform desktop app using Tauri with native installers for macOS, Windows, and Linux
- Automated release pipeline via GitHub Actions publishing versioned assets to GitHub Releases

## Technologies
Tauri, Rust, JavaScript, GitHub Actions
