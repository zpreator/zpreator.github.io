# Moovmetrics - Fitness Tracking

## Description
Comprehensive full-stack fitness tracking application integrating Strava API with Flask backend and SQLite database. Features secure user authentication, interactive data visualizations, and responsive mobile design.

## Details
- Built comprehensive full-stack fitness tracking application integrating Strava API with Flask backend and SQLite database
- Implemented secure user authentication, interactive data visualizations, and responsive design for optimal mobile experience

## Technologies
Flask, SQLite, JavaScript, HTML/CSS, Strava API
