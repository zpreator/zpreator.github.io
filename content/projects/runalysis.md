# Runalysis

## Description
Personal running form analyzer that uses YOLO pose/point tracking on treadmill footage to extract gait metrics and identify bad form

## Details
- Processed treadmill MP4 footage using YOLO for pose estimation and point tracking
- Extracted biomechanical metrics including cadence, stride length, and ground contact patterns
- Trained Random Forest model to classify running form types

## Technologies
Python, YOLO, OpenCV, NumPy
