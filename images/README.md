# Image Organization Guide

## Folder Structure:
```
/images/
├── profile/           # Your profile photos
│   ├── avatar.jpg     # Main profile picture (300x300px recommended)
│   └── headshot.jpg   # Professional headshot
├── projects/          # Project screenshots and demos
│   ├── project1.jpg   # E-commerce platform screenshot
│   ├── project2.jpg   # Task management app screenshot
│   └── project3.jpg   # Weather dashboard screenshot
├── skills/           # Skill-related icons (if needed)
├── background/       # Background images
└── icons/           # Custom icons or logos

## Recommended Image Specifications:

### Profile Pictures:
- Format: JPG or PNG
- Size: 300x300px (square)
- Max file size: 200KB
- Quality: High resolution but web-optimized

### Project Screenshots:
- Format: JPG or PNG
- Size: 400x250px (16:10 aspect ratio)
- Max file size: 300KB each
- Show the actual interface/result

### General Guidelines:
- Use descriptive filenames (e.g., "ecommerce-dashboard.jpg")
- Keep file sizes small for fast loading
- Use JPG for photos, PNG for graphics with transparency
- Consider using WebP format for better compression (modern browsers)
```
